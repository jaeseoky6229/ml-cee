# AI and Data Analysis in Civil & Envirronmental Engineering (ML-CEE)

This repository contains coursework and projects for the undergraduate course **"AI and Data Analysis in Civil & Environmental Engineering."** The course focuses on applying Machine Learning (ML) techniques to civil engineering domains through practical, hands-on Jupyter Notebook assignments.
<br>  


## 📌 Overview
* **Objective:** Implementing ML algorithms for civil and environmental engineering problems.
* **Format:** Weekly laboratory assignments and a term projects.
* **Tools:** Python, Jupyter Notebook, Scikit-learn, etc.
